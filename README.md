# AuVid


A desktop application that allows users to download videos and audio easily.

---

## 🚀 Features

* Download videos in multiple qualities
* Convert to MP3
* Simple and clean interface

---

## 🛠️ Tech

* Python
* yt-dlp
* Tkinter

---

## ▶️ How to run

```bash
pip install yt-dlp
python auvid.py
```
